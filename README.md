# Socicat

> Socicat，一款多平台的视频发布+互动+AI电子宠物程序，让文案生成、视频发布和互动更加便捷

[![Stars](https://img.shields.io/github/stars/lcc-ops/Multi-publisher)](https://github.com/lcc-ops/Multi-publisher/stargazers)

支持微博、抖音、快手、视频号、哔哩哔哩、YouTube、TikTok 等多个主流平台。

---

## 痛点

作为内容创作者，你是否也遇到过这些烦恼？

- **多平台运营**：同时运营多个平台，重复发布让人崩溃
- **手动上传**：每个平台都要打开网页、登录、上传、填写描述...繁琐耗时
- **效率低下**：每次发布内容要花费大量时间在重复操作上
- **养号困难**：账号需要持续互动才能获得流量，但手动操作费时费力

---

## 解决方案

Socicat 让你**一次编辑，一键发布到所有平台**，并提供**自动化养号**能力，模拟真实用户行为。

### 核心功能

- **多平台发布**：微博、抖音、快手、视频号、哔哩哔哩、YouTube、TikTok 等
- **可视化操作**：直观的图形界面，三步完成发布
- **账号隔离**：独立浏览器环境，数据完全隔离
- **批量发布**：一次编辑，同时发布到多个平台
- **账号矩阵**：支持多个账号管理
- **自动养号**：模拟真实用户行为，支持浏览、点赞、评论、关注、搜索互动
- **定时任务**：支持每日、每周、Cron 表达式三种定时策略，自动执行养号
- **多主题支持**：支持浅色、深色、高对比度、护眼浅色四种主题
- **代理配置**：支持为海外平台配置代理

## 0.4.0 新增功能

### 🆕 内容资源库（独立顶级 Tab）

将原本散落在养号页的"角色管理 / 评论池 / 黑名单"独立为顶级**资源库** Tab,统一管理 persona、模板、AI 生成内容。

- **角色管理 (Persona)**:配置 LLM 角色的人设、语气、回复风格,用于养号和回复任务的 AI 生成
- **评论池 (Pool)**:人工写的回复短语,养号 / 回复任务可选用
- **黑名单 (Blacklist)**:养号任务自动跳过名单内的 UP 主 / 关键词
- **内容模板**:支持 4 字段模板 (主题 / 风格 / 长度 / 自定义要求),命名 + 提示词两段式编辑
- **AI 生成器**:基于模板一键生成文案,生成历史可回溯、可一键应用到发布页
- **应用路由**:在资源库生成 / 选择的模板,可直接传到发布页

### 🆕 侧边栏导航重构

- 顶级 Tab 重新排序:首页 / 互动 / 内容库 / 账号 / 日志 / 资源库
- 日志 Tab 移到内容和账号之间,养号 / 回复 / 发布日志统一在"日志"页查询
- "内容资源库"作为独立顶级 Tab,从养号页中拆分出来

### 🆕 日志增强

- 日志新增"评论内容"列,显示养号 / 回复任务实际发出去的评论文本
- 日志新增"视频 URL"列,直接定位到养号评论过的视频
- 错误详情列宽调整 (400→200),避免占用过多屏幕
- 详情弹窗支持查看最终评论全文 (final_text 字段)

### 🆕 LLM 回复稳定性

- 修复 `_extract_content` 解析 Qwen / DeepSeek 等推理模型时 `<think>` / `` 块剥离失效的问题
- 增加单行 think 块兜底,确保 reasoning_split 失效时仍能正确抽取正文

### 🆕 工程化改进

- **统一项目根路径管理**:新增 `paths.project_root()`,杜绝 `Path(__file__).parent.parent...` 心算 (曾导致内容库 db 加载到空库事故)
- **PyInstaller release 流水线**:`.venv-build` 干净构建 + spec 不写 excludes + COLLECT 带版本号目录,产物从 3.1GB 优化到 269MB

## 0.3.0 新增功能

### 🆕 自动养号（出访任务）

按关键词自动去平台搜视频，浏览、点赞、关注、评论，全部跑一遍。模拟真实用户行为，避免手动操作。

- 支持平台：抖音 / TikTok / 快手 / B 站 / YouTube
- 操作可自由组合（浏览 + 点赞 + 关注 + 评论）
- 任务可复制、可立即执行

### 🆕 自动回复（回复任务）

监控指定账号的评论和通知，匹配关键词后用预置评论池或 LLM 角色自动回复。

- 数据源：评论池（人工写的回复） / LLM 角色（AI 实时生成）
- 调度策略：手动 / 每日定时 / 时间窗口轮询

### 🆕 定时调度

- 三种策略：手动执行 / 每日固定时间 / 时间窗口内间隔轮询
- 任务列表显示下次执行时间，到点自动跑

### 🆕 发布日志

新增"日志"顶级菜单页，每次发布的平台、账号、时间、结果、错误详情全部入 DB，可按业务类型（发布/养号/互动）/ 账号 / 平台 / 天数筛选。错误详情支持双击弹窗查看完整堆栈。
